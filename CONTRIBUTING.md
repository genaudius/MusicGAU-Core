# Contributing to MusicGAU-Core

Thank you for your interest in MusicGAU!

## Intellectual Property Notice
While the source code in this repository is visible for transparency and development purposes, please be aware of the following:

- **Model Weights**: The weights and checkpoints of the MusicGAU models are the private property of Gen Audius LLC and are not included in this repository.
- **Original Stems**: The high-fidelity stems and organic percussion recordings used for training are proprietary and not for public distribution.

## How to Contribute
We welcome feedback and bug reports. However, any contributions to the core engine or training pipeline must adhere to the quality standards set by Gen Audius LLC.
